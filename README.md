  ## Description
  This online e-commerce website has been converted to use Redux to monitor state

  ## Installation
  npm i in the root directory and each sub directory (client and server) will install all required packages

  ## Usage
  To use, run npm seed to populate the store, then npm run develop to test the application locally. Push to Heroku for full useage.
